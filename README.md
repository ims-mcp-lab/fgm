# 라이브 상품 등록 UI 데모

이 프로젝트는 Next.js를 사용하여 라이브 커머스 상품 등록 화면을 구현한 예시입니다. 디자인 시안은 Figma 목업을 기반으로 했으며, 등록 단계 헤더, 상품 정보 입력 폼, 모바일 미리보기 컴포넌트로 구성되어 있습니다.

## 기술 스택
- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS Modules

## 주요 컴포넌트
- `StepHeader`: 등록 단계 진행 상황과 토글 스위치를 표시합니다.
- `ProductForm`: 상품명, 할인률, 설명, 혜택, 채팅 설정 등을 입력할 수 있는 폼입니다.
- `PhonePreview`: 라이브 미리보기 화면을 나타내는 휴대폰 프레임 레이아웃입니다.

## 실행 방법
```bash
npm install
npm run dev
```
